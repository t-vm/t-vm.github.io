对手上电子垃圾状态进行的粗略统计

不幸地，这篇文章已经太监了，让我们为它默哀两个普朗克时间
（还是得打个表才行，这么写用处不大）

眼下

度过了一个充斥着求职琐事的三月和一个极度低效的四月，又在四月底的某天晚上下单了一台从垃圾场来的 SIEMENS NIXDORF Mobile 300之后，我瞥了一眼墙角的若干大纸箱，明白随着手上设备的增多，一些工作势在必行了。

做什么

  1. 打表，统计手上设备的状态
  2. 写这篇文章
  3. 按照状态对所有东西装箱分箱

简单的统计

下面是截至这篇文章发表之时对我手上的全部可以被归类为“vintage”的设备的粗略统计。包括他们的当下状态、数量、意向去向这类基本信息。我会分为无法启动的设备和可以启动的设备这两大类来写。
呃，虽然但是在本文里我连表格都懒得做，令人感叹。

可以启动的设备

  1. IBM ThinkPad T43p，14’’/2.13/2G
    1. 来源：货场
    2. 外观状态：佳，A面类肤质涂层有两道划痕
    3. OS：Debian12
    4. 功能：可以启动，有故障
    5. 故障类型：不认硬盘，可能是南桥问题或者sata2pata的ic脱焊。后者可能性大
    6. 备注：光盘和u盘能认
    7. 预计处理方式：维修
  2. IBM ThinkPad X32/1.7/2G
    1. 来源：货场；拼好机
    2. 外观状态：佳，缺螺丝盖片
    3. OS：windowsXP SP1（IBM OEM）
    4. 功能：全部正常，网卡驱动没打
    5. 预计处理方式：改固态盘
  3. IBM ThinkPad A31p/2.4/2G
    1. 来源：货场&奸商；拼好机
    2. 外观状态：一般，缺螺丝盖片，屏幕轻度发黄
    3. OS：windowsXP SP3（IBM OEM）
    4. 功能：可启动有故障
    5. 故障类型：电芯自锁
    6. 预计处理方式：修电池后保留
  4. ThinkPad W700ds(FVT)/2.4(QX9200)/fx3700m/8G
    1. 来源：货场
    2. 外观状态：差，防滚架一处断裂（显卡风扇处），电池仓有一处缺肉，A壳不可逆老化发白，屏幕偏光膜（内）大量污渍
    3. OS：Windows vista（Lenovo OEM）
    4. 功能：可启动有故障
    5. 故障类型：小键盘损坏（排线断）、Fan ERR（有备件；有时fan1有时fan2，前者9成）
    6. 备注：FVT阶段工程机，单屏幕，无电池
    7. 预计处理方式：飞排线、换屏幕面板，维修后永久保留
  5. 51NB ThinkPad X62/i5 5600U/16G
    1. 来源：二手卖家
    2. 外观状态：差
    3. OS：AOSC
    4. 功能：可启动有故障
    5. 故障类型：thinklight损坏
    6. 备注：服役中
    7. 预计处理方式：更换外壳和屏显后保留
  6. ThinkPad 760EL（太久没动配置忘了）
    1. 来源：二手卖家；拼好机
    2. 外观状态：差
    3. OS：暂无
    4. 功能：可启动有故障
    5. 故障类型：电池健康度耗尽、cmos没电
    6. 备注：镍氢忘拆了！镍氢忘拆了！镍氢忘拆了！
    7. 预计处理方式：喷漆，拆镍氢、修电芯、换装CF card固态盘后保留
  7. TOSHIBA libretto W100（工程机)/i5 520UM/2G
    1. 来源：货场
    2. 外观状态：一般
    3. OS：Windows7（TOSHIBA OEM）
    4. 功能：可启动有故障
    5. 故障类型：电芯自锁、上屏触摸有断触（宽度2-3cm）
    6. 备注：所属工程阶段不明；因散热设计缺陷换成了量产版A壳，原壳装在配件机上
    7. 预计处理方式：装win8、修电芯，为不破坏工程机特殊外观上屏断触问题暂时弃疗；永久保留
  8. TOSHIBA libretto W100/u5400/2G
    1. 来源：二手卖家
    2. 外观状态：极佳
    3. OS：Windows7（TOSHIBA OEM）
    4. 功能：可启动有故障
    5. 故障类型：电芯自锁
    6. 预计处理方式：出售或赠送
  9. TOSHIBA libretto W100/u5400/2G
    1. 来源：货场
    2. 外观状态：一般
    3. OS：无
    4. 功能：可启动有故障
    5. 故障类型：触摸坏、屏线无、硬盘无、电池无
    6. 备注：a壳是工程机的；配件机
    7. 预计处理方式：保留
  10. SONY VAIO PCG-U1/TM5800/256m
    1. 来源：货场
    2. 外观状态：较差：掉漆，缺少脚垫、垫片融化、缺键帽和摇杆
    3. OS：无
    4. 功能：可启动有故障
    5. 故障类型：键盘缺帽、缺硬盘排线、无电池、缺右侧摇杆外观件
    6. 备注：已解体
    7. 预计处理方式：拼好机
  11. SONY VAIO PCG-C1S/mmx266/忘了
    1. 来源：群友交换（用VGN-Z5）
    2. 外观状态：一般，A壳有腐蚀、缺脚垫
    3. OS：Windows98（SONY OEM）
    4. 功能：可启动有故障
    5. 故障类型：无电池、cmos没电
    6. 备注：无电池、无扩展内存
    7. 预计处理方式：找电池；保留
  12. SONY VAIO PCG-C1MR/BP /TM5600/忘了
    1. 来源：货场
    2. 外观状态：一般，屏幕偏光膜酸解
    3. OS：Windows me（SONY OEM）
    4. 功能：全部正常
    5. 备注：有原装恢复盘
    6. 预计处理方式：保留
